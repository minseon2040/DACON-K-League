# ⚽ K리그 경기 내 최종 패스 좌표 예측 AI 모델 개발

### Sports Analytics Competition · Spatio-Temporal Player Tracking · Regression AI

Predicting football players’ future spatial positions from tracking data in a sports analytics AI competition.

## Project Overview

- 선수 트래킹 데이터를 기반으로 특정 시점의 좌표를 예측하는 문제

**Period:** Dec 1, 2025 – Jan 12, 2026  
**Team:** 3 members 
**Objective:** Predict players’ final spatial coordinates based on movement data.

## Modeling

- CatBoost & LightGBM for structured tabular features  
- MLP neural network for nonlinear spatial patterns  
- Distance-aware **triple-bucket weighted ensemble**
